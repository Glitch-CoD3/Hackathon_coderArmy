Media Tag in HTML

Tutorial 05


1. Video Tag

Some key points:

  • The <video> tag is a container for video content.
  • It supports formats like MP4, WebM
  • Attributes like controls, autoplay, loop, muted, and poster are used to manage video behavior.

Syntax:

     <video width="400" controls>
     <source src="movie.mp4" type="video/mp4">
     </video>

2. Audio Tag

Some key points:

  • The <audio> tag is a container for video content.
  • It supports formats like mp3
  • Attributes like controls, autoplay, loop, muted are used to manage audio behavior.

Syntax:

     <audio width="400" controls>
     <source src="audio.mp3"">
     </audio>
