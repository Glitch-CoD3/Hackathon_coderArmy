4. Class & ID in HTML Tags

  • Class

    In HTML, a class is an attribute you can add to an element to give it a name or group.

    • We use it mostly for styling with CSS or selecting with JavaScript.
    • One element can have multiple classes, and one class can be used on many elements.
    • It helps keep code organized and reusable.

  • ID

    In HTML, an id is an attribute used to give a unique name to an element.

    • Each id must be used only once in a page.
    • It is often used for styling with CSS
    • Every element can have its own unique id.
