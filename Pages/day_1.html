  2. What is HTML

    HTML means HyperText Markup Language. It is the standard language used to create web pages.

  3. Heading Tag

    In HTML, headings are used to make titles or subtitles. There are 6 levels of headings, from <h1> (biggest, most important) to <h6> (smallest, least important).

    Tag Meaning/Priority Typical Use
    <h1> Highest priority / main title Page title, main topic
    <h2> Second level / main sections Section headings under <h1>
    <h3> Third level / subsections Subsections under <h2>
    <h4> Fourth level Smaller subsections
    <h5> Fifth level Minor headings
    <h6> Lowest priority / least important Smallest headings, rarely used

  4. Paragraph Tag ( <p> )

    Purpose: Used to define a paragraph of text.

    Syntax: <p> This is paragraph syntax </p>


  5. Order List

    Used to create a numbered list.

    Example:

    HTML code :
    
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

    Output :

    1. First item
    2. Second item
    3. Third item

  6. Unorder List

    Used to create a list with bullets (instead of numbers).

    Example:

    HTML code :
    <ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

    Output :

    • First item
    • Second item
    • Third item

  7. Ancor Tag

    • Used to create links to other pages, sections, email addresses, or files.

    • Link to another page

      <a href="page_2"> page_2 </a> ..............On webpage -> page_2


    • Link to a section on the same page (using id)

      <h2 id="contact">Contact Us</h2>

      <a href="#contact">Go to Contact Section</a>

    • Link to email

      <a href="mailto:example@example.com">Send Email</a>

    Syntax Example
    <a href="URL">Link Text</a>

    href → the link’s destination (URL, page, section, email, etc.)

    Link Text → the clickable text visible on the page

    <a href="https://www.google.com" target="_blank">Go to Google</a>

    Go to Google

  8. Image Tag

    The <img> tag in HTML is used to display images on a webpage.

    Syntax: <img src="www.google.com/image" alt="image">

    Set Image width, height, title :

    <img src="myphoto.jpg" alt="photo" width="300" height="200" title="Hello, it's me!">>

    Example

    Hove on image you can see the Title of image..
